# 🎓 VVIT College Admissions Chatbot

A college admissions chatbot built with Django and HTML/CSS/JavaScript. Students who want to take admission can ask about fees, placements, hostel, courses, and more through a clean chat interface.

## Features

- Interactive chat interface with quick reply buttons
- Answers questions about Fees & Courses
- Answers questions about Placements & Salary
- Answers questions about Admission Process
- Answers questions about Hostel & Campus Life
- Typing animation for better user experience
- Fully responsive dark-themed UI

## Tech Used

- **Backend:** Python, Django, SQLite
- **Frontend:** HTML, CSS, JavaScript
- **Logic:** Keyword-based FAQ matching
